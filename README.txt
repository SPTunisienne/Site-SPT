SPT — Pack final prêt à publication

Contenu :
- index.html
- style.css
- script.js

Publication :
1. Déposez ces 3 fichiers dans le même dossier sur votre hébergement.
2. Vérifiez que la page d’accueil s’appelle bien index.html.
3. Le site est bilingue FR / EN dans un seul fichier HTML.
4. Le changement de langue fonctionne sans dépendre d’autres pages.

Solutions simples :
- Netlify
- GitHub Pages
- OVH / Ionos / cPanel / FTP

Remarque :
Le formulaire ou les boutons email utilisent mailto:, donc ils ouvrent la messagerie du visiteur.
